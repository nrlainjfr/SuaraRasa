# 🌿 SuaraRasa
### *Addressing Mental Health Awareness*

> **"A safe space to express, reflect, and heal."**  
> SuaraRasa is an Android mobile app designed to support mental health and emotional well-being, built with empathy, privacy, and accessibility at its core.

</div>

---

## 📖 About

Mental health issues are often stigmatised, leaving many individuals without the support they need. **SuaraRasa** bridges that gap by providing a stigma-free digital space where users can express their feelings, access mental health resources, and find support anytime, anywhere.

---

## ✨ Features

### 💬 Inspirational Quotes
Daily uplifting quotes retrieved from an external API (FavQs) to promote emotional well-being. Users can save their favourite quotes and revisit them whenever they need encouragement.

### 📬 Capsule Letter
Write private, reflective letters addressed to your future self. Schedule delivery by date or duration, tag entries by mood emoji, and filter past letters to identify emotional patterns over time.

### 🆘 Emergency Line Care
A curated list of mental health emergency contacts across Malaysia. Includes 24/7 filter, search by state or name, the ability to add custom contacts, and a favourites system for quick access.

### 🎵 Soul Beats
A collection of calming music tracks — instrumental, lofi, and relaxation beats — to reduce stress. Users can upload their own tracks and curate a personalised playlist.

### 🎮 Mini Games
Simple, engaging distraction games (bubble popping & match cards) designed to give users a mental break and help reduce anxiety. Features custom character artwork and background music.

### 🤖 Teman AI-Chat (Ai-man)
An empathetic AI chatbot powered by Google's Gemini AI, programmed specifically for mental health conversations. Ai-man is not a licensed therapist — it is a compassionate companion designed to help users feel less alone.

### 📊 Mood Tracker
Log daily emotional states with mood tags, personal notes, and timestamps. View your full emotional history at a glance, and edit or delete entries as needed to monitor your mental well-being over time.

### 🧠 Mental Health Assessment (DASS-21)
A self-screening tool based on the DASS-21 standard to evaluate stress, anxiety, and depression levels. Results are displayed with a score and category (Good / Moderate / Critical) to encourage users toward appropriate self-care or professional help.

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest version)
- Android SDK 24+
- Google Gemini API Key

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/nrlainjfr/SuaraRasa.git

# 2. Open in Android Studio
File → Open → select the cloned folder

# 3. Add your API key in local.properties
GEMINI_API_KEY=your_api_key_here

# 4. Run the app
Shift + F10 or click ▶ Run
```
